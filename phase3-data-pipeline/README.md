# Phase 3: 데이터 파이프라인

모델 성능의 80%는 데이터가 결정한다. 의료 문서 데이터의 수집, 정제, 합성 데이터 생성까지 실무 파이프라인을 구축한다.

## 학습 목표

- 의료 문서 유형별 어노테이션 전략 수립
- 개인정보 처리 및 비식별화 방법 이해
- 합성 데이터 생성으로 데이터 부족 문제 해결
- VLM 학습에 최적화된 데이터 로더 구현

## 구성

| Chapter | 주제 | 핵심 내용 |
|---------|------|-----------|
| 7 | 데이터 수집 & 정제 | 문서 유형, 어노테이션, 품질 관리, 개인정보 |
| 8 | 합성 데이터 생성 | 템플릿, 노이즈, Pre-printed 시뮬레이션 |
| 9 | 데이터 포맷 & 로더 | Conversation Format, 효율적 로딩 |

## 실습

| 노트북 | 설명 |
|--------|------|
| `01-synthetic-data-generator.ipynb` | 의료 문서 합성 데이터 생성 |
| `01-dataloader-implementation.ipynb` | VLM용 데이터 로더 구현 |

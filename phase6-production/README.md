# Phase 6: 프로덕션 배포

학습하고 최적화한 모델을 실제 서비스로 배포한다. 안정성, 모니터링, CI/CD까지 실무 운영에 필요한 모든 것을 다룬다.

## 학습 목표

- vLLM + FastAPI 기반 서빙 아키텍처 구축
- 24시간 무중단 운영을 위한 안정성 확보
- Prometheus + Grafana 모니터링 설정
- 모델 버전 관리 및 자동 배포 파이프라인 구축
